# 🎵 PJSK (世界计划) 相关功能

本模块功能源自 **HarukiBot / Hakurui**。
> **提示**：目前功能仍在完善中，如遇问题请联系开发者。

## 🔍 歌曲与谱面查询

| 指令 | 说明 | 示例 |
| :--- | :--- | :--- |
| `pjskinfo <曲名>` | 查看歌曲详细信息 (别名: `song`, `musicinfo`) | `song 消失` |
| `pjskbpm <曲名>` | 查看歌曲 BPM | `pjskbpm 消失` |
| `查bpm <数字>` | 查询对应 BPM 的所有歌曲 | `查bpm 200` |
| `谱面预览 <曲名> <难度>` | 查看谱面预览视频 (来源: ぷろせかもえ！) | `谱面预览 消失 master` |

> **难度缩写支持**：
> `ez`(easy), `nm`(normal), `hd`(hard), `ex`(expert), `ma`(master), `ap`(append)

## 🏷️ 别名设置 (需管理权限)

* **设置歌曲别名**: `musicset <昵称> to <歌名>`
* **删除歌曲别名**: `musicdel <昵称>`
* **查看角色别名**: `charalias <昵称>`
* **设置角色群昵称**: `grcharaset <新昵称> to <已有昵称>` (仅当前群有效)

## 📊 玩家信息与绑定

在指令前加上对应服务器前缀即可查询不同服务器信息：
* **国际服**: `en` (如 `en绑定`)
* **台服**: `tw`
* **韩服**: `kr`
* **国服**: `cn` (默认日服不加前缀)

| 指令 | 功能 |
| :--- | :--- |
| `绑定 <id>` | 绑定游戏 UID |
| `sk` | 查询自己当前的排名和分数 (需前100名) |
| `sk <id>` | 查询指定 ID 的排名和分数 |
| `pjskprofile` | 生成个人信息图片 (需先绑定) |
| `逮捕 @某人` | 查看群友的打歌情况 (FC/AP数) |

## 🌍 MySekai (3D家园) 查询

* **分析材料**: `mysekai_analyze` (或 `ms分析`) - 分析现存材料
* **资源地图**: `mysekai_maps <1-4>` (或 `ms地图`) - 1:草原, 2:花田, 3:沙滩, 4:废墟
* **下载照片**: `mysekai照片 <序号>` - 下载在游戏中拍摄的照片

## 🏆 活动与查线

* **时速/日速**: 查看榜线 PT 增长速度 (如 `时速 10` 查看近10分钟速度)
* **分数线 <排名>**: 查看特定排名的分数趋势
* **查房 <排名>**: 查询特定排名的详细信息
* **查活动 <关键词>**: 搜索活动信息 (如 `查活动 25h`)
* **查卡面 <角色>**: 查询角色卡面 (如 `查卡面 miku`)

## ⚖️ 隐私设置

* **不给看**: 禁止他人使用 `逮捕` 查询你的信息。
* **给看**: 允许他人查询 (默认)。
